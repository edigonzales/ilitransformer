# DMAVTYM Alles V1.1 DMAV -> DM01 Full Run

Dieses Bundle transformiert den eingecheckten DMAV-Volldatensatz
`src/test/data/DMAV_Version_1_1/DMAVTYM_Alles_V1_1.xtf` mit den vorhandenen
`dmav-to-dm01`-Profilen in eine DM01-ITF-Datei.

Nicht abgedeckte Baskets sind im Manifest als `exclude` dokumentiert. Es werden
keine Transferdaten oder generierten kombinierten Mappings versioniert.

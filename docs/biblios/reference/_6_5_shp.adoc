==== Eingabe-Optionen

[cols="1,1,3",options="header"]
|===
| Option | Default | Beschreibung
| `class` | *(Pflicht)* | INTERLIS-Klassenname

| `basketId` | `b1` | Basket-ID

| `geometryType` | aus Shape-Typ | `coord`, `polyline`, `surface`

|===

==== Ausgabe-Optionen

[cols="1,1,3",options="header"]
|===

==== Eingabe-Beispiel

[source,ilimap]
----
input shp_src {
  path "daten/gebaeude.shp";
  model "AV";
  format shp;
  option class "AV.Gebaeude";
  option dbfEncoding "ISO-8859-1";
  option geometryType surface;
}
----

==== Ausgabe-Beispiel

  path "output/gebaeude.shp";

  option prj "PROJCS[\"CH1903+ / LV95\",GEOGCS[...]]";
}

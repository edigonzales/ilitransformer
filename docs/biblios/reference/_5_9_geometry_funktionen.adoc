=== Geometry-Funktionen

[cols="1,3,3",options="header"]
|===
| Funktion | Signatur | Beschreibung
| `coordEquals` | `(coord1, coord2, tolerance) ‚Üí boolean` | Zwei Koordinaten innerhalb einer Toleranz gleich?
| `pointOnSurface` | `(surface) ‚Üí coord` | Deterministischer Punkt auf einer SURFACE/MULTISURFACE
|===

[WARNING]
====
`pointOnSurface()` gibt eine WARNING aus, wenn kein Punkt abgeleitet werden kann. Dies tritt bei degenerierten Geometrien auf (z.B. Fl√§che mit Fl√§che = 0).
====

==== Beispiele

[source,ilimap]
----

  // Koordinatenvergleich mit Toleranz
  GleicherPunkt = coordEquals(s.Koord1, s.Koord2, 0.001);
